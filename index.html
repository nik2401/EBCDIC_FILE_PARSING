AMEX File Parsing Project

This project is designed to parse American Express clearing files and process financial transactions efficiently.

Features

  • Data Extraction and Parsing
  • Database Integration
  • Error Logging and Handling
  • Email Notifications
  • Multiprocessing for Performance

How to Use

Clone the repository, configure the required settings in the SetUp.py file, and run the main script to start processing AMEX clearing files.
